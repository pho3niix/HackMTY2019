## MEVN-Stack
Ambiente de desarrollo hecho con MongoDB, Express.js, Vue.js, Node.js

```bash
# Correr Servidor
`npm run server`

# Correr Cliente
`npm run client`

# Correr Cliente y Servidor al mismo tiempo
`npm run dev`

# Contruir para produccion
`npm run build`

# Iniciar el servidor para produccion
`npm start`
```
